# 🇪🇬 Egypt Tourism Data Analysis

> A data analysis project exploring tourism trends in Egypt using Python, Pandas, and Matplotlib.

---

## 📌 Project Overview

This project analyzes Egypt's tourism data to uncover trends in tourist arrivals, tourism revenue, destinations, countries, and seasonal patterns.

The analysis combines multiple dimension tables with the main fact table to generate insights and visualizations that support data-driven decision making.

---

## 🎯 Objectives

✅ Analyze tourist arrivals over time

✅ Track tourism revenue trends

✅ Compare performance across countries

✅ Identify top tourist destinations

✅ Explore quarterly and seasonal tourism patterns

✅ Create meaningful visualizations and insights

---

## 🛠️ Technologies Used

* 🐍 Python
* 📊 Pandas
* 🔢 NumPy
* 📈 Matplotlib
* 📓 Jupyter Notebook
* 📂 Excel

---

## 🗂️ Dataset Structure

The project follows a Star Schema design:

```text
Main_Data          → Fact Table
Dim_Country        → Country Dimension
Dim_Destination    → Destination Dimension
Dim_Date           → Date Dimension
```

---

## 🔄 Data Processing

The analysis includes:

* Data Loading
* Data Cleaning
* Data Transformation
* Table Merging
* Exploratory Data Analysis (EDA)
* Data Visualization
