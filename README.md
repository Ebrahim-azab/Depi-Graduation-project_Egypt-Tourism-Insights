# 📖 Project Overview

Egypt Tourism Insights is a comprehensive Data Analytics project designed to analyze tourism trends in Egypt using historical tourism data.

The project follows a complete analytics workflow, starting from data preparation and cleaning, followed by SQL querying and Python analysis, and ending with interactive dashboards developed in both **Power BI** and **Tableau**.

The objective is to convert raw tourism data into actionable insights that help understand tourism performance, visitor behavior, destination popularity, and revenue trends.

---

# 🎯 Project Objectives

- Analyze tourism trends across different years.
- Identify the countries contributing the highest number of tourists.
- Discover the most visited tourist destinations.
- Measure tourism revenue and growth over time.
- Analyze seasonal tourism patterns.
- Build interactive dashboards for data exploration.
- Support data-driven decision making in the tourism sector.

# 🔍 Key Insights

- Tourism arrivals have experienced significant fluctuations across the years.
- Seasonal trends reveal peak tourism periods during specific months.
- A small group of countries contributes the majority of visitors.
- Certain destinations consistently attract the highest number of tourists.
- Tourism revenue generally follows visitor volume but also reflects spending patterns.
