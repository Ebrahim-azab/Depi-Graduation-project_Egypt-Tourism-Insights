# 🇪🇬 Egypt Tourism Analytics Dashboard (Tableau)

## 📌 Overview
This project is an interactive **Tableau Dashboard** that analyzes Egypt's tourism data to provide insights into tourist arrivals, revenues, travel purposes, transportation methods, destinations, and seasonal trends.

The dashboard helps users explore tourism performance through interactive visualizations and filters, making it easier to identify patterns and support data-driven decisions.

---

## 📊 Dashboard Features

- Total Tourist Arrivals KPI
- Total Tourism Revenue KPI
- Tourist Arrivals by Year
- Tourist Arrivals by Country
- Tourism Revenue by Country
- Tourist Arrivals by Destination
- Tourist Arrivals by Entry/Border
- Tourist Arrivals by Transportation Method
- Tourist Arrivals by Month
- Tourism Revenue by Quarter
- Revenue per Tourist Analysis
- Purpose of Visit Analysis
- Stay Duration vs Purpose Correlation
- Interactive Filters
