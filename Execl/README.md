# 🇪🇬 Egypt Tourism Analytics (Excel Dashboard)

## 📌 Overview

This project presents an interactive **Microsoft Excel Dashboard** for analyzing Egypt's tourism industry. It combines raw tourism data with Pivot Tables, Pivot Charts, Slicers, and KPIs to provide a clear overview of tourism performance across multiple dimensions.

The dashboard enables users to explore trends in tourist arrivals, tourism revenue, destinations, transportation methods, visit purposes, and seasonal patterns through an easy-to-use interface.

---

## 📊 Features

- Interactive Dashboard
- Dynamic KPIs
- Pivot Tables
- Pivot Charts
- Slicers for filtering
- Country Analysis
- Destination Analysis
- Transportation Analysis
- Purpose of Visit Analysis
- Revenue Analysis
- Monthly & Yearly Trends

---

## 📂 Dataset

The workbook includes the following sheets:

- **Main_Data**
- **Competitor_Data**
- **Exchange_Rate_CBE**
- **Dim_Date**
- **Dim_Destination**
- **Dim_Country**
- **Dim_Transport**
- **Dim_Purpose**
- **Dim_Entry_Point**

Additional sheets contain Pivot Tables, reports, and dashboard components.
